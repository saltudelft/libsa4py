## install watchman manually in docker
```
dpkg -i watchman_ubuntu20.04_v2022.12.12.00.deb
apt-get -f -y install
watchman version
```

## run libsa4py with pyre options
```
cd data
libsa4py process --p source --o results --pyre
```
